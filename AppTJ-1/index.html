推荐一下优秀的软件

2020-04-15 3 min read # 软件推荐

1. 7zip - 压缩软件

优点：轻量级压缩软件，压缩率炒鸡高，解压速度也很快，兼容性也很强，Alpha版本都很稳定
缺点：UI太Low，小白不建议上手，图标总感觉回到了Windows 2000时代。。。
下载链接：https://www.7-zip.org/

2. Edge Chromium - 浏览器

不是你电脑的那个，是基于Chromium的那个
优点：占用内存小，在国内是支持同步的，有国产浏览器的大部分优点，有更好的平滑滚动，浏览器内核随Chromium更新，支持所有Chrome的拓展，还有很多，比如集锦
缺点：小恐龙游戏被微软大刀部砍了。。
下载链接：https://www.microsoft.com/zh-cn/edge/
支持最低支持Windows 7平台，macOS也可以用

3. Chrome - 浏览器

这个总不用我多说了吧
下载链接：https://www.google.cn/chrome/

4. VS Code - 编程IDE

优点：免费开源，拥有强大的拓展性
缺点：软件太庞大了。。
下载链接：https://code.visualstudio.com/

5. Notepad++ - IDE加文本

优点：体积小，支持大部分编程语言
缺点：界面太丑，不包含编译器
下载链接：https://notepad-plus-plus.org/

6. IDM - 下载器

优点：下载速度贼快贼快
缺点：不支持BT,付费，UI太丑
我在这就不提供特别版链接了
下载链接：http://www.internetdownloadmanager.com/

7. FDM - 下载器

优点：免费开源，支持BT下载，速度也很快，颜值高
缺点：下载速度不如IDM
下载链接：https://www.freedownloadmanager.org/zh/

8. Bandizip - 压缩软件

优点：UI清爽，代码页在文件名乱码时很有用
缺点：7.0需要付费，免费版有广告和烦人的激活按钮
下载链接：http://www.bandisoft.com/bandizip/

9. Everything - 搜索工具

优点：可以快速寻找文件，体积小
缺点：UI过于原生
下载链接：https://www.voidtools.com/zh-cn/

10. Notepads - 文本编辑

优点：基于Fluent Design设计，对Markdown有很好的支持，可以多标签
缺点：不 知 道 耶
下载链接：（不给，自己到微软应用商店下）

11. QuickLook - 文件预览

这个软件会让Windows像macOS一样空格浏览文件
优缺点我整理不出来
下载链接：（微软商店~~~）

12. Recuva - 数据恢复

优点：有非常优秀的恢复效果，只要没格式化，文件会保留原名，并且对小白非常友好
缺点：已经停更了，UI比较原始
这个软件其实不用付费了，因为也没用
下载链接：https://www.ccleaner.com/recuva
